## Common Tasks

### Logging In

1. **Login Process**
   - Navigate to the application URL
   - Enter your institutional username and password
   - Click "Login" - you'll be redirected to the home page

### Completing a Questionnaire

1. **Starting a Questionnaire**
   - From the Active Questionnaires page, click on any questionnaire
   - Read any instructions provided
   - Begin answering the questions

2. **Answering Questions**
   - **Question Types**: 
     - **Multiple Choice**: Select from predefined options
     - **Custom Answer Only**: Provide your own written response
     - **Mixed**: Choose from predefined options OR provide a custom answer
   - Navigate between questions using provided controls
   - Your progress is saved automatically

3. **Submitting Your Response**
   - Complete all required questions
   - Submit your final response

## Student Workflows

### Accessing Questionnaires

1. **Navigating to Active Questionnaires**
   - Click "Active Questionnaires" in the navigation menu
   - View all questionnaires currently assigned to you

2. **Completing Questionnaires**
   - Follow the steps in [Common Tasks - Completing a Questionnaire](#completing-a-questionnaire)
   - Note: You cannot view your submitted responses

## Teacher Workflows

### Accessing Your Dashboard

1. **Navigation**
   - Click "Overview" to access your teacher dashboard

### Completing Assigned Questionnaires

1. **View Your Assignments**
   - Navigate to "Active Questionnaires" 
   - See questionnaires where you're assigned as a teacher participant

2. **Completing Questionnaires**
   - Follow the steps in [Common Tasks - Completing a Questionnaire](#completing-a-questionnaire)

### Reviewing Questionnaire Results

1. **Accessing Results**
   - Use "Overview" (Teacher Dashboard) to view questionnaire results
   - Or navigate to "Data Compare" for analysis tools

2. **Analyzing Responses**
   - View responses from questionnaires you participated in
   - Compare data across different questionnaires
   - Analyze patterns and trends in the responses

## Administrator Workflows

### Creating Questionnaire Templates

1. **Accessing Template Management**
   - Navigate to "Templates" in the navigation menu
   - Click to create a new template

2. **Building a Template**
   - **Template Information**: Add title and description
   - **Adding Questions**: Create questions with three available types:
     - **Multiple Choice**: Predefined options only
     - **Custom Answer Only**: User-provided responses only
     - **Mixed**: Both predefined options and custom answers allowed
   - **Save as Draft**: Work on templates over time

3. **Finalizing Templates**
   - **Preview**: Test the questionnaire before finalizing
   - **Finalize**: Lock the template to prevent further editing
   - **Copying**: If changes are needed later, copy the template to create a new version

### Managing Active Questionnaires

1. **Creating Active Questionnaires**
   - Navigate to "Active Questionnaires"
   - Select a finalized template
   - Assign both a student and teacher to each questionnaire
   - Create multiple questionnaires and group them if needed

2. **Questionnaire Groups**
   - Group multiple active questionnaires together
   - Generate multiple questionnaires simultaneously
   - Each questionnaire in a group remains separate for responses and results

### System Administration

**Note**: User management and system administration are handled externally through institutional systems (such as Active Directory), not within the NextQuestionnaire application.

1. **User Management**
   - User accounts are managed through external institutional systems
   - Role assignments (Student, Teacher, Administrator) are configured externally
   - Access permissions are synchronized from the institutional directory

2. **Application Management**
   - Monitor questionnaire completion rates through the application interface
   - Template and questionnaire data management within the application
   - Application-specific settings and configuration
